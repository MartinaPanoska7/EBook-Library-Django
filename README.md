# E-library System

## Опис на проектот
E-library System е веб апликација изработена со Django framework за управување со електронска библиотека.

Апликацијата овозможува:
- регистрација и најава на корисници
- додавање нови книги
- преглед на книги по категории
- уредување и бришење книги
- прикачување и преземање PDF документи
- преглед на детали за секоја книга

## Технологии
Проектот е изработен со:
- Python
- Django
- HTML
- CSS / Bootstrap
- SQLite

## Функционалности

### Корисници
- регистрација
- login / logout

### Книги
- додавање книга
- уредување книга
- бришење книга
- преглед на книга

### Категории
- Education
- Fiction
- Science
