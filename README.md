# E-Library – Веб апликација за електронска библиотека

## Опис на проектот

Овој проект претставува веб апликација за електронска библиотека изработена со Django. Целта на апликацијата е да им овозможи на корисниците да пребаруваат, додаваат и управуваат со електронски книги на едноставен и прегледен начин.

Апликацијата поддржува регистрација и најава на корисници, како и додавање, изменување и бришење на книги. Книгите се организирани по категории, што овозможува полесно пребарување и користење на системот.

## Користени технологии

* Python
* Django
* HTML
* CSS
* Bootstrap 5
* SQLite

## Функционалности

### Кориснички профили

* Регистрација на нов корисник
* Најава на постоечки корисник
* Одјава од системот

### Управување со книги

* Додавање нова книга
* Приказ на детали за книга
* Измена на податоци за книга
* Бришење книга
* Преглед на сопствени додадени книги

### Преглед на книги

* Категорија Education
* Категорија Fiction
* Категорија Science
* Преглед на PDF документ

## Изработка

При изработката на проектот беа реализирани следните чекори:

1. Креирање Django проект и апликација.
2. Дефинирање на модел за книги.
3. Креирање форми за внес на податоци.
4. Имплементација на регистрација и најава на корисници.
5. Изработка на HTML шаблони.
6. Поврзување на шаблоните со Django views.
7. Додавање CSS стилови и Bootstrap компоненти.
8. Имплементација на CRUD операции за книги.
9. Конфигурирање на статички фајлови.
10. Тестирање на функционалностите.

## Стартување на проектот

Инсталација на зависности:

```bash
pip install django
```

Извршување на миграции:

```bash
python manage.py migrate
```

Стартување на серверот:

```bash
python manage.py runserver
```

Апликацијата ќе биде достапна на:

```text
http://127.0.0.1:8000/
```
